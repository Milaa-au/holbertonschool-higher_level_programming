Test de la classe MyList
=========================

Importation du module
>>> import sys
>>> import os
>>> sys.path.insert(0, os.path.abspath('..'))
>>> MyList = __import__('1-my_list').MyList

Test 1: Création d'une liste vide
>>> my_list = MyList()
>>> print(my_list)
[]

Test 2: Ajout d'éléments
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Test 3: Affichage de la liste triée
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Test 4: Vérification que la liste originale n'est pas modifiée
>>> print(my_list)
[1, 4, 2, 3, 5]

Test 5: Liste avec des nombres négatifs
>>> my_list2 = MyList()
>>> my_list2.append(-5)
>>> my_list2.append(3)
>>> my_list2.append(-1)
>>> my_list2.append(0)
>>> my_list2.append(2)
>>> print(my_list2)
[-5, 3, -1, 0, 2]
>>> my_list2.print_sorted()
[-5, -1, 0, 2, 3]

Test 8: Liste vide
