Basic usage
===========

    >>> matrix_divided([[2, 4], [6, 8]], 2)
    [[1.0, 2.0], [3.0, 4.0]]

    >>> matrix_divided([[1, 2]], 3)

    >>> matrix_divided([[1.5, 3.0]], 2)

Different divisors
=================

    >>> matrix_divided([[1, 2], [3, 4]], -2)
    [[-0.5, -1.0], [-1.5, -2.0]]

    >>> matrix_divided([[1, 2], [3, 4]], 2.5)
    [[0.4, 0.8], [1.2, 1.6]]


Different shapes
================

Single element:

    >>> matrix_divided([[10]], 5)
    [[2.0]]

Single row:

    >>> matrix_divided([[1, 2, 3, 4]], 2)
    [[0.5, 1.0, 1.5, 2.0]]

Single column:

    >>> matrix_divided([[1], [2], [3]], 2)
    [[0.5], [1.0], [1.5]]


Matrix immutability
==================

    >>> m = [[1, 2, 3]]
    >>> matrix_divided(m, 1)
    [[1.0, 2.0, 3.0]]
    >>> m
    [[1, 2, 3]]


Edge numeric cases
=================

Zeros:

    >>> matrix_divided([[0, 0], [0, 0]], 5)
    [[0.0, 0.0], [0.0, 0.0]]

Infinity divisor:

    >>> matrix_divided([[1, 2]], float('inf'))
    [[0.0, 0.0]]
